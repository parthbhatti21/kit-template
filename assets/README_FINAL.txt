🔧 CUSTOMIZATION GUIDE
═════════════════════════════════════════════════════════════════════════════════

TO CHANGE PROPERTY DETAILS:
1. Open index.html in text editor
2. Find and replace:
   • "567 Kit Lane" → Your address
   • "92654" → Your zip code
   • "450000" → Your price
   • "2 Bed" → Correct bedrooms
   • "2 Bath" → Correct bathrooms
   • "1,200 SF" → Correct square footage
   • "2020" → Correct year built

TO CHANGE AGENT INFO:
1. Find "Contact Agent" section in footer
2. Update:
   • Agent name
   • Phone number (tel: link)
   • Email address (mailto: link)
   • DRE License number

TO CHANGE COLORS:
1. Find color hex codes:
   • #1a3d4d (navy primary)
   • #4a7c8c (teal secondary)
   • #a8c5d1 (light blue accent)
2. Replace with your preferred colors throughout

TO UPDATE MAP LOCATION:
1. Find "propertyLocation" in map script section
2. Update latitude (lat) and longitude (lng)
3. Update attractions coordinates if needed

TO CHANGE CALCULATOR DEFAULTS:
1. Find "value=" in mortgage calculator section
2. Update:
   • mortgage-price: 450000
   • mortgage-down-percent: 20
   • mortgage-rate: 6.5
   • mortgage-term: 30
